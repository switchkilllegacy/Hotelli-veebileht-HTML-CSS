# Hotelli-veebileht-HTML-CSS

Lihtne mitmeleheline veebileht, tehtud HTML ja CSS abil.

## Lehed

- **index.html** – avaleht (menüü, vitriin, info, võimalused)
- **about.html** – kirjeldus ja info
- **contact.html** – kontakt ja vorm

## Teemad

- HTML navigatsioon (menüü)
- CSS kujundus
- failiteed (nt `css/style.css`)
- HTML vormid
- HTML5 semantilised märgendid
- mobiilivaade (responsive design)

## Struktuur

website/
├── css/
│ └── style.css
├── img/
├── index.html
├── about.html
└── contact.html
